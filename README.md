# 🚨 Fraud Detection and Risk Analysis in Financial Transactions

## 📌 Project Overview
This project focuses on identifying fraudulent transactions using SQL and Power BI. It analyzes transaction data to detect suspicious patterns and high-risk activities.

## 🎯 Objectives
- Detect fraudulent transactions
- Identify high-risk customers
- Analyze fraud trends
- Build interactive dashboard

## 🛠 Tools Used
- MySQL
- Power BI
- Excel

## 📊 Dataset
- Total Records: 50,000
- SQL Validation: 10,000 rows
- Fields: Transaction ID, Customer ID, Amount, Time, Location, Payment Method, Fraud Flag

## 🔍 Key Features
- Fraud detection using SQL queries
- Risk level classification (High, Medium, Low)
- KPI metrics (Fraud %, Total Transactions)
- Interactive dashboard with slicers

## 📈 Dashboard Insights
- Fraud trends over time
- Fraud by location
- Fraud by payment method
- High-risk customers
